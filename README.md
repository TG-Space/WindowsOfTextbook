# 教材之窗 - 新旧教材收集整理站
> 新旧教材资源归档与对比查阅平台

## 项目简介

**教材之窗**是一个面向广大师生、家长及教育工作者的开源项目，专注于**收集、整理、归档各学段、各版本新旧教材**，统一规范分类，方便快速检索、对照查阅，助力教学、备课、学习与教研。

## 项目特色

- 🌱 **覆盖全面**：收录小学、初中、高中等多学段的新旧版教材
- 📂 **分类清晰**：按学科、出版社、年代版本规范整理，结构一目了然
- 🚀 **持续更新**：随教材版本迭代持续补充以便完善资源
- 🤝 **开源共享**：欢迎社区共同维护，让资源更完整、更易用

## 使用说明

本仓库按「学段 → 学科 → 出版社→版本」进行分类存放，可直接在对应目录中查找所需教材文件。
如需新增、纠错或完善资源，欢迎提交 Issue 或 Pull Request。

## 适用人群

- 学生：预习、复习、对照新旧知识点
- 教师：备课、教研、对比教材变化
- 家长：辅导孩子、查找对应版本教材
- 教育爱好者/内容创作者：学习、研究、素材参考
